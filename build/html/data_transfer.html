1. Data Transfer to BioCommmonsÂ¶

Genome Annotation uses CloudStor as a data transfer portal for file sharing. CloudStor is a file sharing and cloud storage solution for the research and education sector. Please follow the instruction below to upload your data (e.g. reference genome, annotation files and tracks)

  1. Fill up on-board form
  2. A CloudStor URL will be sent to you for data upload
  3. Prepare your reference genome, annotation file and 2bit file.

File Format Web Apollo accepts reference genome in FASTA, 2bit and GFF3 file format.

  • FASTA format is a text-based format for sequences
  • 2bit format stores multiple DNA sequences in a compact randomly-accessible format.
  • GFF3 format is the annotation file of reference genome.
