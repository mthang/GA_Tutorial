3. Create Reference GenomeÂ¶

Reference genome can be made available on web apollo with two options.

  1. upload reference genome using Jbrowse
  2. contact Biocommons user support

Upload Reference genome using Jbrowse. Please follow the instructions below.

  1. prepare your reference genome (e.g FASTA, GFF3 and 2bit)
  2. obtain an account from BioCommons team
  3. logon to web apollo using Jbrowse
  4. following the data_loading tutorial to upload your reference genome.
